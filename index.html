    딥러닝 기반 Java 프로그램 자동 코딩 가이드 모델
    Taeha Hong, Honguk Woo
    Dec 2020
    Abs HTML Code

    대규모 소프트웨어 제품을 개발할 때, 프로젝트팀 구성원들 사이에서 공통적인 코딩 가이드라 인과 표준을 갖는 것이 프로젝트 코드 관리에 중요한 이슈가 된다. 이 논문에서는 팀원들이 공 통의 코딩 가이드라인에 맞추어 개발을 진행하는데 도움이 되는, 딥러닝 기반 자동 코딩 가이 드 모델을 제안한다. 제안된 모델은 Java로 작성된 메소드 단위 코드에 대하여, AST(abstract syntax tree)를 이용하여 여러 코드 형태를 생성하고, 이를 RNN(recurrent neural network)을 통 해 학습하여, 해당 메소드에 logging과 exception handling의 필요 여부를 예측한다. 이 논문에서 는 3가지 Java 기반 오픈소스 프로젝트 코드 데이터로 case study를 수행하여 예측 성능 정확도 를 측정하였다.
